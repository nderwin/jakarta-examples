# Spring Boot Example

Created based on the [Spring Boot](https://spring.io/guides/gs/spring-boot/) documentation.

## Build

```
mvn clean package
```

## Run

```
java -jar target/spring-boot.war
```

## REST Endpoint

```
http://localhost:8080/resources/ping
```

## OpenAPI

```
http://localhost:8080/v2/api-docs
```
